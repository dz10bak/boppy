# Boopy

Aplikacja do wysyłania emoji z Apple Watcha. Landing page promujący aplikację.

## Struktura projektu

- `index.html` — cała strona w jednym pliku (HTML + CSS + JS inline)

## Sekcje strony

1. **Hero** — nagłówek, mockup Apple Watcha z emoji, CTA do App Store
2. **Features** — 6 kart z funkcjami aplikacji
3. **Voice Clone (Coming Soon)** — zapowiedź funkcji klonowania głosu z formularzem waitlist
4. **Footer** — linki, branding

## Konwencje

- Projekt to single-file HTML — cały CSS i JS trzymamy inline w `index.html`
- Ciemny motyw (#0a0a0a tło)
- Font: Inter (Google Fonts)
- Animacje: wyłącznie CSS (bez zewnętrznych bibliotek)
- Design: minimalistyczny, responsywny
